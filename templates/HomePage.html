Using - Instruction

How to use this system for depression detection is divided into the following three steps.

Step1：上传所需的文件

（1）上传语音、文本文件各，分别对应正面、负面、中性问题的回答
（2）语音文件格式(.wav)
（3）文本文件格式(.txt)

Step2：多模态分析

（1）文本处理：Bert预训练模型
（2）语音处理：Mel谱图+NatVlad
（3）映射至公共语义空间：
   Bi-GRU(Attention)+Resnet18(CBAM)
（4）多模态融合：复数化胶囊网络

Step3：返回测试结果

（1）返回抑郁or正常测试结果
（2）提供被试者分布定位图像
（3）个体可视化分析：词云图、SDS分数变化图、正负情绪概率图等
